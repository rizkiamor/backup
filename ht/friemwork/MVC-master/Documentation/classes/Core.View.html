Provides a nice interface to call view helpers.

__call(string $helperName, array $param) : string

This is a magic function, so any calls to the view/view helper which do not exist will end up here. We only pass through the first parameter to make for a nicer implementation in each view helper. This is why it needs to be an array.

access public
magic
