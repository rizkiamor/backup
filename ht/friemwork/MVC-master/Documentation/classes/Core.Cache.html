Interface for creating, getting, and removing items from the cache.

Sample usage:

if (Core\Cache::has('foo')) {
    echo Core\Cache::get('foo');
} else {
    $var = 'Hello World!';
    Core\Cache::put('foo', $var);
    echo Core\Cache::get('foo');
}
