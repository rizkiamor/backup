Whilst in controller context, move to another controller/action.

forward(string $action, string $controller) 

When forwarding to a new action or controller/action, the URL will not change, this is an internal redirect.

Forwarding to an action in the same controller

$this->forward('newAction');

Forwarding to a new controller and action.

$this->forward('newAction', 'newController');
access public

Parameters

$action

string

The action we wish to forward to.

$controller

string

The controller we wish to forward to.

Exceptions

\Core\Exception From the Dispatcher if the controller/action does not exist.

Redirect the user to a new page.

redirect(string $url) 

Unlike the forward() function, this will perform a header redirect, causing the URL to change.

access public

Change the layout from the default.

setLayout(boolean | string $layout) : boolean

If you do not want your View Script to use a layout then pass in boolean false. To set a new layout then pass in the layouts file name without the .phtml extention. Your layouts are located in your MyProject/Layout directory. For example:

$this->view->setLayout('default');
access public

Parameters

$layout

booleanstring

Which layout we wish to use.

Exceptions

\Core\Exception If the layout does not exist.

boolean
